Database Schema

Tables

  • Exercise Table
    • Exercise ID
    • Exercise Name
    • Foreign Key - Muscle ID
  • Muscle Table
    • Muscle ID
    • Muscle Name
  • Routine Table
    • Routine ID
    • Routine Name
  • Routine/Exercise Table - Link the routine to its exercise
    • Routine ID
    • Exercise ID
  • Days table
    • Day ID
    • Exercise ID -Foreign key
    • Date Lifted
  • Sets Table
    • Sets Id
    • Day ID -Foreign key
    • Reps
    • Weight
  • Max Table - Record the users maxes
    • Max Id
    • Max Weight
    • Day ID -Foreign key
    • Sets ID - Foreign Key
    • Exercise Id - Foreign Key

Server Side API

ORM

  • Exercise Object
    • create(name, muscleID)
    • getAllExercises()
    • getExerciseById(id)
    • updateExercise(id, newName, muscleId)
    • deleteExercise(id)
  • Muscle Object
    • create(name)
    • getAllMuscles()
    • getMuscleById(id)
    • updateMuscle(id, newName)
    • deleteMuscle(id)
  • Routine Object
    • create(name)
    • getAllRoutines()
    • getRoutineById(id)
    • getExercisedInRoutine(id)
    • deleteRoutine(id)
  • Day Object
    • create(date)
    • getDatesAndSets(exerciseID) // Use SQL join to get all sets for every date for selected exercise
    • getDate(id)
  • Set Object
    • create(dayid, reps, weight)
    • getSetsByDay(dayId)
    • updateSet(id, reps, weight)
    • deleteSet(id)
